1.  Vaya a _Hospital \> Patients_ y cree un paciente.
2.  Rellene la fecha de nacimiento; el campo _Age_ se calcula automáticamente.
3.  Asigne etiquetas desde _Patient \> Tags_ para clasificar pacientes.
4.  Cree citas desde _Hospital \> Appointments_ y añada líneas con los
    productos/servicios asociados.
